## Быстрая настройка

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте модель в .env файле

Создайте файл `.env` в корне проекта (если его нет) и добавьте:

```env
# Путь к вашей модели GPT-OSS-120B
GPT_MODEL_PATH=C:\Users\vmare\path\to\your\model.gguf

# Тип модели (llama-cpp для GGUF формата)
GPT_MODEL_TYPE=llama-cpp

# Параметры (опционально)
GPT_N_CTX=4096
GPT_N_THREADS=4
GPT_N_GPU_LAYERS=0
```

**Важно:** Укажите правильный путь к вашей модели. Для Windows используйте прямой путь или двойные обратные слеши.

### 3. Запустите сервер

```bash
python manage.py runserver
```

### 4. Откройте страницу AI анализа

Перейдите по адресу: `http://localhost:8000/ai-analysis/`

Или используйте меню "AI Анализ" в навигации (для админов).

## Использование

### Анализ данных

1. На странице AI анализа нажмите "Запустить анализ"
2. (Опционально) Введите вопрос для анализа
3. Дождитесь результата

### Генерация графиков

1. Нажмите "Сгенерировать график"
2. AI предложит оптимальный тип графика
3. График будет отображен автоматически

## Проверка статуса модели

На странице AI анализа отображается статус модели:
- ✅ **Готово к работе** - модель загружена и готова
- ⚠️ **Не настроена** - укажите GPT_MODEL_PATH в .env
- ❌ **Ошибка** - проверьте путь к модели и логи

## Подробная документация

См. файл `AI_SETUP.md` для подробной информации о настройке различных типов моделей.

## Поддерживаемые форматы моделей

1. **llama-cpp** (GGUF) - рекомендуется для CPU
2. **transformers** (PyTorch) - для GPU ускорения
3. **vLLM** - для больших моделей через API

## Требования

- Python 3.8+
- Модель GPT-OSS-120B скачана на компьютер
- Достаточно RAM для загрузки модели (для 120B требуется много памяти)
